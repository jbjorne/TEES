waitForProcess(process, numCorpusSentences, measureByGap, outputFile, counterName, updateMessage, timeout=None)

source code 

Waits for a process to finish, and tracks the number of entities it writes to it's outputfile. If writing a sentence takes longer than the timeout, the process is considered stalled and is killed.
